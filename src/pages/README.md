# Pages globales

Ce dossier contient uniquement les pages front-end qui ne sont pas encore rattachées a une feature dediee.

Règle :

- garder ici les pages transverses ou temporaires ;
- deplacer dans `src/features/<feature>/pages/` des qu'une page appartient clairement a une feature ;
- ne pas ajouter de code metier partage ici.
