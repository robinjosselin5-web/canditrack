# Sprint Goal

Mettre en place les fondations techniques de CandiTrack en développant le système complet d'authentification.

## Objectifs

- Création de compte
- Connexion
- Déconnexion
- Réinitialisation du mot de passe
- Gestion du profil

## Résultat attendu

À la fin du Sprint, un utilisateur peut créer un compte, s'authentifier, gérer son profil et accéder aux fonctionnalités protégées de l'application.
